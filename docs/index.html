🔐 Secure

Built-in support for private and presence channels with authentication.

Prometheus metrics for comprehensive monitoring and alerting.

🐳 Docker Ready

Quick Start

# Pull and run from Docker Hub
docker pull funal/soketi-rs:latest
docker run -d -p 6001:6001 -p 9601:9601 funal/soketi-rs:latest

# Or use Docker Compose
git clone https://github.com/ferdiunal/soketi.rs.git
cd soketi.rs/deployment/docker
docker-compose up -d
